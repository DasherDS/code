### React 中的`ECharts` 优化

1. 数据处理优化
   - 控制数据量，如对大量数据进行聚合、抽样或分页处理。
   - 优化数据格式，保证简洁且符合要求，尽早处理需转换的数据。
2. 组件渲染优化
   - 合理使用 `React.memo` 避免不必要的重新渲染
   - 避免 `ECharts` 实例的不必要重新创建
3. `ECharts` 配置优化
   - 按需配置，只启用需要的功能和组件，关闭不必要的默认功能。
   - 优化动画，适当降低帧率，考虑渐进式或无动画的数据更新方式
4. 内存管理优化
   - 组件卸载时及时释放定时器、事件监听等资源
   - 防止因与其他组件或库交互导致的内存泄露
