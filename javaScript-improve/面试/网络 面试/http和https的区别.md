### `http`和`https`的区别

主要的区别在于安全性和数据传输方式上，`https`比`http`更加安全，适合用于保护网站用户的隐私和安全。

* 安全性：`http` 协议传输的数据都是未加密的，也就是明文的，因此使用`http`协议传输的数据可以被任何抓包工具截取并查看。而`http`是协议是有`ssl + http` 协议构建的可进行加密传输、身份认证的网络协议，更为安全。
* 数据传输方式：`http`协议的端口号是80，`https`协议的端口号是443
* 网址导航栏显示：`http://`    `https://`
* 证书：`https`需要到CA申请证书，一般免费证书较少，因而需要一定费用。
* 网络速度：`http`协议比`https`协议快，因为`https`协议需要进行加密和解密的过程
* `SEO`优化：搜索引擎更倾向于把`https`网站排在更前面的位置，因为`https`更安全
